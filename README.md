# 🌾 Smart Agro — Premium Agriculture Platform

Smart Agro is a modern, responsive web application designed for an organic farming and agricultural solutions provider. The platform showcases sustainable farming methods, natural product categories (Fruits & Vegetables), and agricultural project portfolios with a premium, user-friendly UI.

---

## 🎯 Key Features

* **Premium Agri-UI:** Clean layout with optimized visual hierarchies, dynamic sections, and fully responsive design.
* **Product Showcase:** Segmented display for organic items like Apples, Oranges, Garlic, Carrots, and more.
* **Multi-Page Architecture:** Includes structured routes for comprehensive user navigation:
  * **Home:** General introduction, market impact stats, and featured highlights.
  * **About Us:** Core mission statement focusing on sustainable agriculture.
  * **Our Products:** Detailed view of available organic categories.
* **Modern Asset Optimization:** Utilizes high-performance image optimization layers via Next.js routing.

---

## 💻 Tech Stack

* **Framework:** Next.js
* **Styling:** Tailwind CSS
* **Icons & Fonts:** Optimized Google Fonts & UI icons
